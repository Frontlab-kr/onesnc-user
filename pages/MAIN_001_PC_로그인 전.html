개인정보처리방침
CODEF API 도입으로
다양한 경영정보 제공이 가능해졌어요.
진형세무회계의 경정청구 자동화 서비스 ‘택스리서치’의 주요 기능인 ‘국세 및 지방세 납부내역을 조회’하는데 API 사용이 필수적이라는 것을 알게 되었습니다. 헥토데이터를 통해 필요한 API 대부분을 적용할 수 있었고, 고객사의 니즈에 적절히 대응해 준다는 추천 후기에 헥토데이터의 CODEF API를 이용하게 되었습니다.

지난 5년간 과다 납부한 국세와 지방세를 자동으로 산출하기 위해서는 10개가 넘는 세금 관련 API 연결이 필요했습니다. 헥토데이터는 API 상품이 상세하고 다양하게 준비되어 있어서 필요한 기능을 손쉽게 구현할 수 있었습니다.

 기존에는 환급금을 확인하려면 복잡한 서식과 여러 가지 신고 서류를 세무회계 전문가에게 전달해야 했지만, 이제는 택스리서치에서 환급금 확인부터 경정청구 위임 의뢰까지 한 번에 할 수 있습니다. 인증을 거치면 헥토데이터 API를 통해 고객이 홈택스 또는 위택스에 신고한 국세, 지방세 관련 데이터를 안전하게 전송받을 수 있고, 전송받은 데이터는 진형세무회계에서 다년간 축적한 경정청구 노하우을 담은 연산식을 거쳐 예상 환급금으로 산출되어 고객 화면에 나타납니다. 금액이 연산되는 30초 남짓 동안 고객들은 자사와 비슷한 매출 규모를 가진 회사가 1년간 평균 얼마의 세금을 감면받았는지 함께 확인할 수 있습니다.
